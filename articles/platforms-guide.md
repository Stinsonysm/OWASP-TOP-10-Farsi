#### مثال Lab:

```
Lab: SQL injection vulnerability in WHERE clause

Mission:
سایت یه فیلتر دسته‌بندی داره.
باید با SQL Injection همه محصولات رو نمایش بدی!

URL: https://lab.web-security-academy.net/filter?category=Gifts

Steps:
1. Intercept با Burp
2. تغییر category به: Gifts' OR '1'='1--
3. همه محصولات نمایش داده میشه
4. Lab حل شد! ✅

Explanation:
Query اصلی:
SELECT * FROM products WHERE category = 'Gifts'

بعد از Injection:
SELECT * FROM products WHERE category = 'Gifts' OR '1'='1'--'
                                              ↑ همیشه درسته!
```

#### سطوح Lab:

```
🟢 Apprentice  - مبتدی
🟡 Practitioner - متوسط
🔴 Expert      - پیشرفته
```

## 💻 ماشین‌های مجازی آسیب‌پذیر

### 7. DVWA (Damn Vulnerable Web Application) 🎯

**معروف‌ترین وب‌سایت آسیب‌پذیر!**

#### نصب:

```bash
# با Docker (راحت‌ترین!)
docker run --rm -it -p 80:80 vulnerables/web-dvwa

# یا دانلود:
git clone https://github.com/digininja/DVWA.git
```

#### سطوح امنیت:

```
🟢 Low      - خیلی آسیب‌پذیر
🟡 Medium   - یکم محافظت
🔴 High     - محافظت بیشتر
⚫ Impossible - امن (برای مقایسه)
```

#### آسیب‌پذیری‌ها:

```
✅ Brute Force
✅ Command Injection
✅ CSRF
✅ File Inclusion
✅ File Upload
✅ SQL Injection
✅ XSS (Reflected)
✅ XSS (Stored)
✅ Weak Session IDs
```

#### مثال تمرین:

```
Vulnerability: SQL Injection
Level: Low

Code:
$query = "SELECT * FROM users WHERE id = '$id'";

Exploit:
1. Input: 1' OR '1'='1
2. Query: SELECT * FROM users WHERE id = '1' OR '1'='1'
3. همه کاربرا! ✅
```

**امتیاز:** ⭐⭐⭐⭐⭐ (5/5) - Must have!
