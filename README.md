## 📚 فهرست مطالب

### [۱. کنترل دسترسی شکسته 🚪🔓](articles/01-broken-access-control.md)
وقتی کاربرا می‌تونن به جاهایی برن که نباید! مثل اینکه مهمون توی مهمونی بره تو اتاق خوابت! 😱

**چیزایی که یاد می‌گیری:**
- چطور هکرا به داده‌های بقیه دسترسی پیدا می‌کنن
- نحوه محافظت از بخش‌های حساس سایت
- مثال‌های واقعی و بامزه

---

### [۲. شکست رمزنگاری 🔐💔](articles/02-cryptographic-failures.md)
وقتی رمزنگاری ضعیفه یا اصلا نیست! مثل فرستادن نامه عاشقانه روی کارت پستال! 💌

**چیزایی که یاد می‌گیری:**
- چرا باید داده‌ها رو رمز کنی
- چطور پسورد رو درست ذخیره کنی
- تفاوت HTTP و HTTPS

---

### [۳. تزریق (Injection) 💉😈](articles/03-injection.md)
خطرناک‌ترین آسیب‌پذیری! وقتی هکر می‌تونه کد مخرب توی سیستم تو اجرا کنه!

**چیزایی که یاد می‌گیری:**
- SQL Injection چیه و چطور کار می‌کنه
- انواع Injection (Command, NoSQL, etc.)
- نحوه جلوگیری با Prepared Statements

---

### [۴. طراحی ناامن 🏗️💥](articles/04-insecure-design.md)
وقتی از اول طراحی سیستم غلطه! مثل ساختن خونه بدون در! 🚪❌

**چیزایی که یاد می‌گیری:**
- تفاوت باگ و طراحی بد
- Threat Modeling چیه
- اصول طراحی امن

---

### [۵. پیکربندی نادرست امنیتی ⚙️🔧](articles/05-security-misconfiguration.md)
وقتی همه چی درست نصب شده ولی تنظیماتش غلطه! مثل داشتن ماشین گرون با در باز! 🚗

**چیزایی که یاد می‌گیری:**
- تنظیمات پیش‌فرض خطرناک
- Security Headers
- چک‌لیست قبل از Deploy

---

### [۶. کامپوننت‌های آسیب‌پذیر و قدیمی 📦🦕](articles/06-vulnerable-outdated-components.md)
وقتی از کتابخونه‌های قدیمی استفاده می‌کنی که باگ امنیتی دارن!

**چیزایی که یاد می‌گیری:**
- چطور کتابخونه‌ها رو آپدیت نگه داری
- npm audit و Snyk
- Supply Chain Attacks

---

### [۷. شکست شناسایی و احراز هویت 🎭🔑](articles/07-identification-authentication-failures.md)
وقتی سیستم لاگین ضعیفه! مثل قفل با کلید "۱۲۳۴"! 🔓

**چیزایی که یاد می‌گیری:**
- پسورد قوی
- Multi-Factor Authentication (2FA)
- Session Management
- Brute Force Protection

---

### [۸. شکست یکپارچگی نرم‌افزار و داده 📦🔓](articles/08-software-data-integrity-failures.md)
وقتی نمی‌دونی فایلی که دانلود کردی دستکاری شده یا نه!

**چیزایی که یاد می‌گیری:**
- Code Signing
- Subresource Integrity (SRI)
- Insecure Deserialization
- Supply Chain Security

---

### [۹. شکست لاگ‌گیری و نظارت امنیتی 📹🙈](articles/09-security-logging-monitoring-failures.md)
وقتی هک میشی ولی خبر نداری! مثل بانک بدون دوربین! 🏦

**چیزایی که یاد می‌گیری:**
- چی رو باید لاگ بگیری
- چطور Alert بذاری
- تشخیص حملات
- SIEM و Monitoring

---

### [۱۰. جعل درخواست سمت سرور (SSRF) 🎭🌐](articles/10-server-side-request-forgery.md)
وقتی هکر از سرور تو برای حمله استفاده می‌کنه!

**چیزایی که یاد می‌گیری:**
- SSRF چیه و چطور کار می‌کنه
- دسترسی به AWS Metadata
- نحوه جلوگیری با Whitelist
